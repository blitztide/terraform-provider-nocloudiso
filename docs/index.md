# NoCloudIso Provider

This provider for [Terraform](https://www.terraform.io/) / [OpenTofu](https://opentofu.org/) is used for generating ISO files to provision VMs using [Cloud-Init](https://cloudinit.readthedocs.io/en/latest/reference/datasources/nocloud.html) or [CloudBase-Init](https://cloudbase-init.readthedocs.io/en/latest/config.html#config-drive)

This provider allows for the creation of arbitrary [ISO9660](https://en.wikipedia.org/wiki/ISO_9660) files which contain custom files.
